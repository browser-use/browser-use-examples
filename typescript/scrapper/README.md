## Developer Guide

```sh
# Generate Schema Migrations
pnpm drizzle-kit generate

# Set Up Development Database
docker compose up -d

# Migrate Schema
pnpm drizzle-kit migrate

# Start Development Server
pnpm dev

# Start Browser Use CLI
pnpm webhook
```
